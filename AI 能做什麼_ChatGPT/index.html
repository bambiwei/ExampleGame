AI 能做什麼

一句話版本

你提出教學目標，我協助把想法變成可用的教材、活動、通知、表單、網頁或自動化流程。

1. 我是誰，結合 Antigravity 後能做什麼

我是 ChatGPT

我擅長理解需求、整理脈絡、生成文字、設計流程、解釋概念，也能把複雜內容改寫成學生、家長或同事容易理解的版本。

加上 Antigravity

我可以在本機工作區建立檔案、編輯網頁、檢查專案結構、執行指令與測試，讓「幫我做一份東西」從文字建議變成實際成品。

2. 我能做的事與教學日常例子

備課

課程設計與教案

例：幫國小五年級自然課設計「水溶液酸鹼」40 分鐘教案，含引起動機、分組實驗、提問與評量。

評量

出考卷與題庫

例：依據課本範圍產出選擇、填充、簡答題，附答案、配分、難度與雙向細目表。

教材

講義、簡報與學習單

例：把一篇閱讀文本做成分層學習單，包含詞語解釋、理解題、延伸討論與補救題。

班經

班級經營

例：設計開學第一週班規共創活動、座位調整原則、值日生制度與正向回饋語句。

溝通

家長聯絡與通知

例：把「孩子最近作業常遲交」改寫成溫和、具體、可合作的親師溝通訊息。

差異化

分層與補救教學

例：同一個數學概念設計基礎、標準、挑戰三種任務，讓不同程度學生都能參與。

行政

表格、紀錄與報告

例：整理導師會議紀錄，產出待辦清單、負責人、期限與可貼到公告的摘要。

活動

校內活動企劃

例：規劃閱讀週活動流程，含海報文案、闖關任務、志工分工與備品清單。

數位

網頁與互動小工具

例：做一個班級任務抽籤器、計時器、單字練習頁或成果展介紹頁，直接輸出 HTML。

語言

翻譯與改寫

例：把英文通知翻成自然中文，或把正式公文改成學生看得懂的說明。

回饋

學生作品評語

例：根據作文內容產生具體、有鼓勵也有下一步建議的評語，避免空泛稱讚。

研究

資料整理與閱讀摘要

例：摘要一份教育文章，整理成三個可用於教學現場的重點與實作建議。

3. 老師最該裝的 Antigravity 插件 / 擴充功能

瀏覽器預覽 / Live Preview

做班網、教材頁、互動練習時，可以立刻看成果，不必一直猜畫面長什麼樣。

Markdown Preview

適合寫教案、會議紀錄、班級公告與學習單草稿，邊寫邊確認排版。

PDF 工具

方便查看教材 PDF、講義、考卷與行政文件，也能配合我做摘要與重點整理。

Excel / CSV 檢視器

處理成績、問卷、點名與分組名單時更直覺，我也能協助清理欄位與產生統計。

Prettier 或格式化工具

若要做網頁或簡單程式，格式化工具會讓檔案乾淨好維護，交給同事也比較不痛苦。

Git / 版本控制工具

適合長期累積教材與班級網站，能記錄每次修改，改壞時也比較容易找回來。

4. 怎麼跟我說，回答會最好

Do 1：說清楚對象、年級與目標

例如：「幫國中七年級做 20 分鐘英語口說活動，目標是練習過去式。」

Don't 1：只說「幫我做教材」

範圍太大時，我會先猜；你給越多教學限制，成品越貼近現場。

Do 2：指定格式與可用時間

例如：「請用表格列出，活動總長 40 分鐘，材料只能用白板和便利貼。」

Don't 2：把所有任務混在一句話

最好分步驟：先做大綱，再做題目，最後請我檢查語氣與難度。

Do 3：提供範例或既有資料

貼上課文、題型、班級狀況或舊講義，我可以沿用你的風格與需求。

Don't 3：期待第一次就完全定稿

把我當共同備課者：先產草稿，再請我「更口語、降低難度、增加活動性」。

Do 4：要求我自我檢查

例如：「完成後請檢查是否符合素養導向、是否有答案、是否有時間分配。」

Don't 4：直接貼敏感個資

學生姓名、電話、身分證字號等請先匿名化，用「學生 A」「家長 B」就很夠用。

你可以把我當成一位耐心、會整理、也能動手做檔案的教學協作夥伴。
